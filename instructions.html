Uploading Instructions

Classes

  1. Open Google Calendar on a computer.
  2. Create a new calendar to add your schedule to.
  3. On the left side, click > Import calendar
  4. Click Choose File and select the downloaded file. The file should end in .csv
  5. Choose the calendar you created earlier to add the imported events to.

    Warning: By default, events will be imported into your primary calendar, making them difficult to delete or change later.

  6. Click Import.

Community Periods

  1. Open Google Calendar on a computer.
  2. Copy: https://calendar.google.com/calendar/ical/friendsseminary.org_t3ndfap268hhcj4tq1kdqi1440%40group.calendar.google.com/public/basic.ics
  3. On the left side, click > Add by URL
  4. Click Choose File and select the downloaded file. The file should end in "csv"
  5. Paste the link.
  6. Click Add Calendar.
