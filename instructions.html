Uploading Instructions

  1. Open Google Calendar on a computer.
  2. Create a new calendar to add your schedule to.
  3. In the top right, click > Settings.
  4. Open the Calendars tab.
  5. Click Import calendars between the "My calendars" and "Other Calendars" sections.
  6. Click Choose File and select the file that was downloaded. The file should end in "csv"
  7. Choose the calendar you created earlier to add the imported events to.

    Warning: By default, events will be imported into your primary calendar, making them difficult to delete or change later.

  8. Click Import.
